ИНСТРУКЦИЯ ПО УСТАНОВКЕ SYSTEMD СЕРВИСА
========================================

ВАЖНО: Твоя версия не требует модификаций!
  - systemd запустит npm, который найдет node и ts-node сам
  - Логи пишет systemd через journalctl

1. Сначала убей текущий процесс если он запущен:
   ps aux | grep "ts-node.*index.ts" | grep -v grep
   kill <PID>

2. Отредактируй drugs-bot.service файл:
   - Поменяй User=ubuntu на твоего юзера (если не ubuntu)
   - Поменяй WorkingDirectory на путь к проекту  
   - ОБЯЗАТЕЛЬНО поменяй /path/to/node и /path/to/ts-node на реальные пути!
   - Поменяй пути к логам если надо

3. Скопируй сервис файл в системную папку:
   sudo cp drugs-bot.service /etc/systemd/system/

4. Перезагрузи systemd:
   sudo systemctl daemon-reload

5. Запусти сервис:
   sudo systemctl start drugs-bot

6. Включи автозапуск при старте системы:
   sudo systemctl enable drugs-bot

КОМАНДЫ ДЛЯ УПРАВЛЕНИЯ:
=======================

Проверить статус:
  sudo systemctl status drugs-bot

Посмотреть логи:
  sudo journalctl -u drugs-bot -f

Остановить:
  sudo systemctl stop drugs-bot

Перезапустить:
  sudo systemctl restart drugs-bot

Отключить автозапуск:
  sudo systemctl disable drugs-bot

Посмотреть логи из файла:
  tail -f /home/ubuntu/drugs/app.log
